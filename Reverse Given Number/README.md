# Reverse Given Number

## Description
This program contains a simple logic that reverses a given number. It takes an integer as input and returns the reversed version of that number.


## Examples

### Example 1
- **Input:** `12345`
- **Output:** `54321`

### Example 2
- **Input:** `-9876`
- **Output:** `-6789`

### Example 3
- **Input:** `100`
- **Output:** `1`

### Example 4
- **Input:** `7`
- **Output:** `7`
